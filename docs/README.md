# Documentation

## Structure

| Directory | Description |
|-----------|-------------|
| `docs/` | Project-specific documentation (SRS, SDD, etc.) |
| `docs/common/` | Shared documentation for all application projects |

## Project-Specific

Documentation unique to this project:

- `formal/` - SRS, SDD, STP for this specific project
- `diagrams/` - Project-specific architecture diagrams

## Common (Shared)

Documentation shared across all application projects via submodule:

- `guides/` - Development guides, best practices
- `templates/` - Document templates
- `diagrams/` - Common architectural patterns

**View online:** [https://abitofhelp.github.io/hybrid_app_docs/](https://abitofhelp.github.io/hybrid_app_docs/)
