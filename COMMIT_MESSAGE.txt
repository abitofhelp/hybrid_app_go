## Migration Guide

### For Result[T] usage:
```go
// Before (samber/mo)
import "github.com/samber/mo"
result := mo.Ok[string]("value")
if result.IsOk() {
    v := result.MustGet()
}

// After (custom domain)
import domerr "github.com/abitofhelp/hybrid_app_go/domain/error"
result := domerr.Ok[string]("value")
if result.IsOk() {
    v := result.Value()
}
```

### For domain testing:
```go
// Before (testify)
require.True(t, result.IsOk())
assert.Equal(t, want, got)

// After (stdlib only)
if !result.IsOk() {
    t.Fatalf("expected Ok, got Error: %v", result.ErrorInfo())
}
if got != want {
    t.Errorf("got %v, want %v", got, want)
}
```
